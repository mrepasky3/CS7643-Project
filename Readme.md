# Project for Deep Learning Class

Three datasets for set-based classification problems:  
1. Toy Data  
  * Sum over digits  
  * Max over digits  
  * Max - Min of digits  
  * Lowest-valued mode of set of digits  
  * Product of two largest digits in a set  
2. MNIST Data  
  * Sum over digits  
  * Max over digits  
  * Max - Min of digits  
  * Lowest-valued mode of set of digits  
  * Product of two largest digits in a set  
3. Omniglot Data  
  * Number of unique symbols in the set
